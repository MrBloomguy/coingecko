## 기능
- Firebase 데이터베이스를 사용하여 회원 가입/로그인 기능
- CoinGecko API를 사용하여 가장 인기 있는 암호 화폐와 트렌드 통화
- CoinGecko API를 사용
- 특정 코인을 선택하여 관심 목록에 추가
- 각 코인의 정보/가격/변경 사항이 있는 지정된 페이지
- 다크 모드 / 라이트 모드

반응형 웹사이트(Tailwind 사용)</br>
hambuger menu CSS를 이용한 모바일 웹사이트
